 GET请求：

POST 表单请求,与GET方式一致,采用KEY-VALUE方式标识键值对:

                            
fullcoudivonId=115&shodivId=123&adivdivCont=0


                            
POST JSON请求,直接将JSON内容作为请求参数，值得注意的是，CONTENT-TYPE需要选择与之相对应的:

                            
{"link":{"charset":"UTF-8","protocol":"http","method":"post","name":"本机提交接口","contentType":"application/json","url":"127.0.0.1:8080/link/add.do"}}


                            
 如果需要添加请求头,使用3个%%%分割请求参数和请求头,而多个请求头则使用3个&&&符号分割

                            
fullcoudivonId=115%%%User-Agent=Mozilla/5.0...&&&Cookies==https://m.yunjiglobal.com/yunjibuyer/static/vue-buyer/idc/index.html#/sure-order...
